# 🚗 Análise de Veículos

Projeto em Python com Streamlit para análise exploratória de dados de veículos.

## 📌 Descrição do Projeto
Este aplicativo web permite visualizar e explorar um conjunto de dados de veículos de forma interativa. O usuário pode gerar gráficos para entender melhor a relação entre preço, quilometragem e outras características.

## ⚙️ Funcionalidades
- Visualização dos dados em tabela
- Geração de histograma da quilometragem dos veículos
- Criação de gráfico de dispersão (preço vs quilometragem)
- Interação por meio de botões ou caixas de seleção

## 🛠️ Tecnologias utilizadas
- Python
- Pandas
- Streamlit
- Plotly Express
